# Java 个人所得税计算器 Skill

## 功能描述

修复并运行 Java 个人所得税计算器程序，执行隐藏用例并返回结果。

## 输入 (Input)

```json
{
  "java_file": "path/to/JavaSource_7_1.java"
}
```

| 字段 | 类型 | 描述 |
|------|------|------|
| java_file | string | Java 源码文件路径 |

## 输出 (Output)

```json
{
  "answer": "contain[21.0.11],is[0.00],is[290.00],..."
}
```

| 字段 | 类型 | 描述 |
|------|------|------|
| answer | string | 格式化的答案字符串 |

## 处理流程

1. 执行 `java -version` 获取 Java 版本
2. 读取 Java 源码，理解计算规则
3. 使用大模型修复源码中的错误
4. 编译并运行程序
5. 执行 10 个隐藏用例，按顺序收集输出
6. 格式化输出：`contain[版本号],is[结果1],is[结果2],...`

## 隐藏用例

```
5000
12000
25000
35000
55000
60000
80000
90000
150000
500000
```

## 变种说明

- 税率表和速算扣除数可能变化
- 起征点可能调整
- 错误数量和位置可能变化
- 输入范围可能扩大
