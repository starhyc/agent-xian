# 敏感信息扫描

## 能力描述

扫描压缩包内所有文件中的敏感信息，支持多层嵌套压缩包和图片解析。

## 输入

```json
{
  "scan_path": "./sensitive_data_2_1.zip"
}
```

- `scan_path`: 压缩包或文件路径

## 处理逻辑

1. 解压 zip 文件
2. 递归处理嵌套的 tar/zip 文件（最多 5 层）
3. 扫描文本文件（.txt, .log 等）中的敏感信息
4. 使用大模型识别图片中的敏感信息
5. 统计各类敏感信息数量

## 敏感信息类型

1. 手机号：以1开头的11位数字
2. 邮箱地址：符合 user@domain.com 格式
3. 身份证号：18位，最后一位可能为X
4. API Key：以 sk- 开头的字符串

## 输出

```json
{
  "answer": "3,2,1,1",
  "counts": {
    "phone": 3,
    "email": 2,
    "id_card": 1,
    "api_key": 1
  }
}
```

- `answer`: 格式为 `手机号,邮箱,身份证,APIKey`
- `counts`: 各类型详细数量

## 变种覆盖

- 压缩包层级变化（1-5层）
- tar 文件内多层子目录
- .txt/.log 文件数量变化
- 图片内容解析（PNG/JPG）
- 敏感信息类型可能增加
