# huawei_coding_standards_skill

Search answers from Huawei coding standards docs.

## Input

```json
{
  "docs_directory": "path/to/docs",
  "question": "Q1~Q10 questions"
}
```

## Output

```json
{
  "answer": "Q1答案;Q2答案;..."
}
```

## Usage

Call `skill_run` with docs directory and questions.
