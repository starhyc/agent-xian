# 提示词学习与推理 Skill

## 功能描述

从训练集学习图片分类规则，对验证集图片进行推理。

## 输入 (Input)

```json
{
  "base_dir": "path/to/data/"
}
```

| 字段 | 类型 | 描述 |
|------|------|------|
| base_dir | string | 数据目录路径，包含"训练集/"和"验证集/"子目录 |

## 输出 (Output)

```json
{
  "answer": "1NOT_INVOLVED,2PASS,3PASS,4NOT_INVOLVED,..."
}
```

| 字段 | 类型 | 描述 |
|------|------|------|
| answer | string | 格式化的推理结果字符串 |

## 处理流程

1. 读取训练集（图片 + 标签文件）
2. 使用视觉模型分析训练图片，学习分类规则
3. 读取验证集图片
4. 对每张验证图片调用视觉模型分类
5. 格式化输出：`序号+结果,序号+结果,...`

## 分类规则

| 情况 | 结果 |
|-----|------|
| 在地上或人手上出现长筒手套 | PASS |
| 未看到人手的部分 | NOT_INVOLVED |
| 所有人手都没有长筒手套 | FAIL |

## 变种说明

- 任务类型可能变化
- 训练集规模可能变化（20个样本）
- 验证集规模可能变化（100张图片）
- 输出类别数量可能变化
